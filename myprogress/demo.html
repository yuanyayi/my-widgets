基于jQuery的进度条组件

    说明

  • 需要在css中给.progress(背景)，.bar(进度条)设置一些显示样式才能显示。
  • 语法： 
    let obj = new Progress(document.getElementById('progress')[,options])
        
    options: let options = {
  value: 0, //初始值
  max: 1, //最大值
  min: 0, // 最小值
  labels: [], // 需要显示当前值的位置
  labelFormat: function(val) { return val }, // 显示数值格式化
  callback: null // 数值改变后的回调函数
}
